# AI CRM HCP Interaction Logger

## Overview
AI-powered CRM application to log and manage Healthcare Professional (HCP) interactions.

## Tech Stack
- Frontend: React
- Backend: FastAPI
- Database: MySQL
- AI: LangGraph + Groq LLM

## Features
- AI-based interaction extraction
- Save interaction
- Search interactions
- Edit interactions
- Interaction summary
- Follow-up recommendations

## Installation

### Backend
```bash
cd BACKEND
pip install -r requirements.txt
